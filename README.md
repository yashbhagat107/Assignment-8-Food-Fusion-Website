# Food-Fusion | Premium Food Ordering App

**Food-Fusion** is a responsive React application designed to simulate a real-world e-commerce food delivery platform. It leverages modern web technologies like Redux Toolkit for state management, Axios for API integration, and Lazy Loading for optimized performance.

## Key Features

* **Dynamic Menu:** Fetches real-time food data from TheMealDB API using **Axios**.
* **Smart Cart System:** Managed via **Redux Toolkit**, allowing users to add items, update quantities, and calculate totals instantly.
* **Performance Optimized:** Implements **Lazy Loading (React.lazy & Suspense)** to load pages only when needed, ensuring fast initial load times.
* **Responsive Design:** Fully responsive UI built with CSS Grid and Flexbox, providing a seamless experience on Mobile, Tablet, and Desktop.
* **Navigation:** SPA navigation using **React Router DOM v6**.

## Tech Stack

* **Frontend:** React.js
* **State Management:** Redux Toolkit
* **Routing:** React Router DOM
* **API Client:** Axios
* **Styling:** CSS3 (Variables, Flexbox, Grid)
* **Icons:** React Icons
